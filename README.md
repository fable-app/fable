# Fable
**Learn German through beautiful stories**

A React Native app with Japanese minimalist aesthetics for learning German by reading bilingual short stories.

Fable embodies Japanese minimalist aesthetics - inspired by the soft, inviting feeling of premium Japanese stationery.

- Generous breathing room
- Soft, muted colors
- Subtle animations
- Focus on reading comfort
